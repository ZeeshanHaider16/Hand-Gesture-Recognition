# Hand-Gesture-Recognition
Project Overview

This project implements a real-time Hand Gesture Recognition system using computer vision and machine learning techniques. The system detects hand landmarks from live video input and classifies gestures to enable intuitive human–computer interaction. It demonstrates practical applications of AI in interactive systems, automation, and touchless control interfaces.

🚀 Features

Real-time hand detection and tracking

Landmark extraction using MediaPipe

Gesture classification pipeline

Smooth video processing with OpenCV

Extensible architecture for adding new gestures

🛠️ Tech Stack

Programming Language: Python

Libraries: OpenCV, MediaPipe, NumPy

Concepts: Computer Vision, Landmark Detection, Real-time Inference

⚙️ Installation & Setup

Clone the repository

git clone https://github.com/ZeeshanHaider16/Hand-Gesture-Recognition.git

cd Hand-Gesture-Recognition

Install dependencies

pip install -r requirements.txt

Run the project

python main.py
